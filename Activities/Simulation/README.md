# ATN Activities: Simulation

Using ATN to explore scenarios and projected outcomes under defined assumptions.

This activity can be part of the Operations workflow.

## Sources

Representative sources include:

- [NASA Systems Engineering Handbook](https://www.nasa.gov/wp-content/uploads/2018/09/nasa_systems_engineering_handbook_0.pdf), which supports evaluating scenarios and behaviors before or during operational decision-making.
- [DoD Systems Engineering Guidebook](https://www.cto.mil/wp-content/uploads/2024/05/SE-Guidebook-Feb2022.pdf), which uses analytical and scenario-based evaluation across development and operations.
- [SEBoK: Applying Life Cycle Processes](https://sebokwiki.org/wiki/Applying_Life_Cycle_Processes), which discusses iteration between problem understanding, solution exploration, and later operational use.
