# ATN Examples: Theories

Examples of ATN specifications for theories.

## [Truth](./Truth)

A minimal ATN specification for a theory with a single proposition.

## [Implication](./Implication)

A minimal ATN specification for a theory with a logical dependency between two propositions.

## [Ordering](./Ordering)

A minimal ATN specification for a theory with an abstract ordering relation over a type.

## [Combination](./Combination)

A minimal ATN specification for a theory with an abstract function, an identity element and algebraic laws.

## [Taxonomy](./Taxonomy)

A minimal ATN specification for a theory with a subtype taxonomy and classification constraints.
